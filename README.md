## 慕课网秒杀系统实现

### 包及类的功能
|包| 功能|
|:----:|:----:|
|entity|数据库表对应的实体类|
|dao|data access object|
|service|实现service接口和实现类|
|exception|service接口所需要的一些异常|
|dto|用于web层和service层进行数据的传输|
